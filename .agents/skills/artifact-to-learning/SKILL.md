# Artifact-to-Learning Skill

Use when a method has worked repeatedly and should become durable repo knowledge.

Instructions:

1. Read `.agents/loops/artifact-to-learning.md`.
2. Require evidence from at least two distinct cases unless the rule is a high-severity safety rule.
3. Keep learning scoped, non-obvious, and reviewable.
4. Update or mark stale any contradictory old lesson.
